# 🧠⚡ Pokedex AI Web App

An AI-powered Pokémon recognition web application that identifies Pokémon from images and displays their stats using a deep learning model.

---

## 🚀 Live Demo
👉 Deployed on Render (link coming soon)

---

## 🎯 Features

✨ Upload Pokémon image  
🤖 Predict Pokémon using trained TensorFlow model  
📊 Display stats (HP, Attack, Defense, Speed)  
⚡ Fast and interactive UI  
🧠 Deep Learning powered classification  

---

## 🛠️ Tech Stack

- 🐍 Python
- 🌐 Flask
- 🤖 TensorFlow / Keras
- 📊 Pandas & NumPy
- 🎨 HTML, CSS, JavaScript
